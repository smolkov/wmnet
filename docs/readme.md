## Install node.js und yarn

```bash
# ubunty
sudo apt install nodejs
```

## Install vuepress

```bash
# install
yarn global add vuepress
# OR npm install -g vuepress
```

## Serve and build

```bash
# start writing
vuepress dev

# build to static files
vuepress build
```
